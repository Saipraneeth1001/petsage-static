# PetSage Portal

Minimal public-facing website for **PetSage**, built with **Next.js App Router**, **TypeScript**, and **Tailwind CSS**.

## Tech
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Editable Content
Primary copy lives in:
- `content/siteContent.ts`

Update text there without touching component code.

## Local Development
Prereqs:
- Node.js 18+ (recommended 20+)

1) Install deps:
   - `npm install`
2) Run:
   - `npm run dev`

Open `http://localhost:3000`.

## Deploy to Vercel
1) Push this repo to GitHub (or import into Vercel).
2) Deploy.
